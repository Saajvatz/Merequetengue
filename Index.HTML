Sistema de codigo abierto

El software de código abierto es software desarrollado y mantenido a través de una colaboración abierta, y disponible, normalmente de forma gratuita, para que cualquiera pueda utilizarlo, examinarlo, modificarlo y redistribuirlo como prefiera.

Sistema de codigo Cerrado

cuando el código fuente no se encuentra disponible para cualquier usuario, es decir no se hace público. Se le llama así en contraposición al código abierto.
